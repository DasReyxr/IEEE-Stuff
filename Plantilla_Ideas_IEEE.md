# Plantilla — Ideas / Proyectos (Rama Estudiantil IEEE UAA)

Úsala para proponer concursos, alianzas, talleres o iniciativas. La idea es que cada propuesta quede clara en 1 página y sea fácil de aprobar/ejecutar.

## Cómo usar
1. Copia/pega la sección **Ficha de idea** por cada propuesta.
2. Llena lo mínimo indispensable (problema, objetivo, responsable, fecha tentativa).
3. Si se aprueba, define **Próximos pasos (1 semana)**.
